assets/
=======

Place your visual assets here.

Suggested files (replace placeholders inside index.html):

  main-visual.png         Main hero visual / logo (Home)
  event-window.png        Preview screenshot for Event Window project
  silver.png              Preview screenshot for Silver Dashboard project
  sectors.png             Preview screenshot for Sectors project
  sentiment.png           Preview screenshot for Sentiment project

To use main-visual.png in the Home, replace the
<div class="visual-placeholder">...</div> block in index.html with:

  <img src="assets/main-visual.png"
       alt="Perspectiva Matemática" class="visual-placeholder" />

Recommended max width for hero image: 760px.
Recommended aspect ratio: 1:1 or 4:3.
